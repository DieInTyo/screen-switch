# Screen Switch

Небольшое Windows tray-приложение, которое по клику на иконку в области уведомлений переносит окна между двумя мониторами.

## Что делает

- левый клик по иконке в трее переносит все обычные окна между двумя мониторами;
- относительное положение и размер окна сохраняются относительно рабочей области монитора;
- у иконки есть контекстное меню: повторный перенос и выход;
- если мониторов не два, приложение покажет предупреждение.

## Запуск

```powershell
dotnet build
dotnet run
```

После запуска приложение появится в скрытых значках Windows.
